# Review Process

## When Review Is Required

Review is required for:
- new foundational platform choices
- security-sensitive technology choices
- new databases, identity providers, or messaging platforms
- production third-party SaaS with company data
- deviations from enterprise standards

## Review Inputs

A review request should include:
- business context
- proposed choice
- alternatives considered
- security implications
- operational model
- cost implications
- migration plan
- fallback or exit strategy
