# Technology Standards

## Baseline Standards

- Use Linux-based runtimes for most server workloads.
- Prefer containerized deployment for new services and applications.
- Prefer PostgreSQL for relational database workloads unless a documented exception applies.
- Prefer standards-based identity and federation protocols such as OIDC, OAuth 2.0, and SAML.
- Prefer REST for synchronous service APIs and event-driven messaging for asynchronous integration.
- Prefer Infrastructure as Code for production environments.
- Use centralized logging and metrics collection for all production systems.
